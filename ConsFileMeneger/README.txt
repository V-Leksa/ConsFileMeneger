Программа консольный файловый менеджер

Файловый менеджер начального уровня предоставляет базовые операции для управления файлами и каталогами, а также обеспечивает интерфейс для навигации и выполнения операций.

В поле, где указан путь к корневой директории компьютера, вводятся команды.
Если операция успешно выполнена, то сообщение об этом появится в окне - "Операция успешно выполнена" (кроме тех операций, которые должны вернуть результат в информационное окно).

Функционал по командам:

menu - в консоли выводится список всех команд для навинации и выполнения операций

cd - перейти в домашнюю директорию
Пример команды:cd название папки (название папки должно быть без пробела, допускается нижнее подчеркивание)

cd..- переход на уровень выше

ls - вывод списка папок и файлов по заданной странице
Пример команды: ls - номер страницы (чтобы вывести список, необходимо перейти в папку командой cd)[ls - 4]

ls -l - детальная информация о файлах и папках
Пример команды:ls -l название файла.txt (название файла должно быть без пробела, допускается нижнее подчеркивание, чтобы появилась информация, нужно находиться в папке, где находится файл или папка, команда cd)

cp - копирование файла или папки
Пример команды: cp название файла/папки

mkdir - создание новой папки
Пример команды: mkdir название папки (название папки должно быть без пробела, допускается нижнее подчеркивание)

touch - создание нового пустого файла
Пример команды: touch название файла.txt (название файла должно быть без пробела, допускается нижнее подчеркивание)

mv - перемещение или переименование файлов или каталогов
Пример команды: mv старое имя файла.txt новое имя файла.txt (название файла должно быть без пробела, допускается нижнее подчеркивание)
Пример команды: mv старое имя папки новое имя папки (название папки должно быть без пробела, допускается нижнее подчеркивание)

rm - удалить файл
Пример команды: rm название файла

rmdir - удалить пустую папку
Пример команды: rmdir название папки (название папки должно быть без пробела, допускается нижнее подчеркивание)

rm-r - удалить директорию и ее содержимое
Пример команды: rm -r название папки (название папки должно быть без пробела, допускается нижнее подчеркивание)

cat - вывести содержимое файла
Пример команды:cat название файла

red - открыть файл в текстовом редакторе
Пример команды: red название файла.txt (название файла должно быть без пробела, допускается нижнее подчеркивание)

echo - передать текст в указанный файл
Пример команды:echo название файла.txt (название файла должно быть без пробела, допускается нижнее подчеркивание) текст, который нужно передать в файл

head - вывести первые 10 строк файла
Пример команды:head -f название файла (название файла должно быть без пробела, допускается нижнее подчеркивание),будет выводить ошибку так как название команды без f

tree - вывод дерева
Пример команды: tree номер страницы путь

Предупреждение! Команда может не выполниться с первого раза. В этом случае следует повторно осуществить действия до получения сообщения об успешном выполнении операции.

saveOfCommands - сохранение вводимых команд
exeptions - сохранение ошибок при выполнении программы
